Research Experience

Research Intern at Yale University
  • June 2017 - Present; Advisor: Minlan Yu
  • Building a performance diagnosis system for NFV chains.
    • Build a user-oriented on-demand performance diagnosis system for NFV chains; users write declarative language to locate faults in NFV chains when they notice performance anomalies.
    • Develop a SQL-like language to support a variety of user-defined queries.
    • Utilize a per-batch monitoring method to minimize performance impact to existing NFV chains.
  • Designing generic modules to enable fast and accurate measurement in software.
    • Investigated various algorithms for different network measurement tasks, and found that filtering out numerous small flows helps improve their accuracy.
    • Designed a generic module, named cold filter, to efficiently perform such filtering operation (part of this work was done at Peking University).
    • Presented a novel packet-aggregating method assisted by modern SIMD instructions to improve the processing speed of existing measurement algorithms.
Research Intern at Peking University
  • April 2016 - March 2017; Advisor: Tong Yang
  • Improving the accuracy and performance of flow size estimation in software.
    • Proposed two data structures (the ABC and Pyramid sketches) to automatically and dynamically allocate memory for each flow in real time, achieving high accuracy for flow size estimation.
    • Proposed to constrain the multiple hashed counters of each flow into one machine word to reduce the cost of both memory accesses and hash computations.
    • Proposed the One Memory Access sketch to do automatic and dynamic memory allocation with two memory accesses per packet in the worst case, guaranteeing performance stability.
